# Finance

> Stock research and analysis — from pre-market to post-trade.

Personalized stock research workflows: pre-market briefing, intraday tracking, post-trade review. Tailored to personal holdings and investment strategies.

## Scope

- **Pre-market briefing** — overnight news, opening signals, key levels
- **Intraday tracking** — real-time alerts, volume spikes, momentum shifts
- **Post-trade review** — what happened, why it happened, what to do better

All data originates from public sources. Analysis is for personal reference only — not financial advice.
